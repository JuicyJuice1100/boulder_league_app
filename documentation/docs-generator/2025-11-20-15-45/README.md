# Boulder League App Documentation

## Overview

Boulder League App is a Flutter mobile application for managing boulder climbing leagues. It enables climbers to track their scores across different gyms, seasons, and weeks while competing on leaderboards.

## Architecture Summary

### Technology Stack
- **Frontend**: Flutter/Dart (v3.35.7)
- **Backend**: Firebase (Firestore, Authentication)
- **State Management**: Custom InheritedWidget Provider pattern
- **Form Handling**: flutter_form_builder

### Core Features
1. **User Authentication** - Email/password login with Firebase Auth
2. **Gym Management** - Create and manage climbing gyms
3. **Season Management** - Define seasons with start/end dates per gym
4. **Boulder Tracking** - Track boulders per gym, season, and week
5. **Score Recording** - Log attempts and completion status for boulders
6. **Leaderboards** - View rankings based on accumulated scores

### Data Model Relationships
```
Gym (1) --> (N) Season
Season (1) --> (N) Boulder
Boulder (1) --> (N) ScoredBoulder
User (1) --> (N) ScoredBoulder
```

### Scoring System
- Completed boulder: 100 points base
- Flash bonus (1 attempt): +25 points
- Multiple attempts: -0.1 points per attempt
